Quickstart Guide

Installation

To get started with AIRA, you'll need to install the AIRA CLI. Open a terminal and run the following command:

npm install -g aira-cli

This will install the AIRA CLI globally on your system.

Project Setup

Create a new directory for your AIRA project and navigate to it:

mkdir my-aira-project
cd my-aira-project

Initialize a new AIRA project in the directory:

aira init

This command will set up the necessary files and configurations for your AIRA project.

Model Deployment

Once your project is set up, you can deploy your AI model to the AIRA network. Make sure your model file (e.g., model.onnx) is in the project directory, then run:

aira deploy model.onnx

AIRA will handle the deployment process, distributing your model across the network of executors.

Model Execution

To execute your deployed model and make predictions, use the AIRA API:

import aira

client = aira.Client()
result = client.execute("model.onnx", input_data)
print(result)

The AIRA client will handle the communication with the network and return the model's predictions.
